# 🍊 枇杷的挂机工坊 (Pipa's IdleForge)

> 欢迎光临！把你的奇思妙想交给我们，挂机就行~

---

## 项目愿景

这是一个**AI 驱动的自动化实验工厂**——但不止于此。这是你的**个人生活操作系统**，是一座充满人情味的「数字领地」。

**核心本质**：将人类的瞬时灵感（低熵）通过 Agent 密集计算（过程熵）转化为高质量的工程产物（高价值低熵）的转换器。

---

## 🍊 世界观设定

### 角色们

| 角色 | 身份 | 职责 |
|------|------|------|
| 🍊 **枇杷** | 工坊主 / 看板娘 | 接待你的奇思妙想，微笑着分发任务 |
| 🧠 **Moss** | 核心动力炉 / 管家 | 计算、调度、监控、确保工坊不停转 |
| 👷 **工人们** | 各工位专业工人 | Researcher、Designer、Developer、Tester... |

### 三种颜色的委托

| 颜色 | 类型 | 范例 | 产物 |
|------|------|------|------|
| 🔴 **红色** | 现实锚点 | 佳福大厦办税务、希腊电表 | Notion Action List |
| 🔵 **蓝色** | 虚拟工厂 | AI 漫剧、RVC 复刻、猫猫视频 | 视频/图像文件 |
| 🟢 **绿色** | 逻辑实验室 | 量化系统、Python 脚本 | Git 代码 + 报告 |

## ✨ 特色功能

### 长期记忆系统
- **情节记忆**：完整任务记录
- **语义记忆**：跨项目知识
- **程序记忆**：避坑指南、工作流模版

### 人类共鸣层
- **首席架构师访谈**：意图对齐
- **技术树决策**：关键节点确认
- **共鸣检查**：阶段性汇报
- **思维链查看**：直接干预

### Moss 大脑
- **异常检测**：死循环、停滞识别
- **资源配置**：自动选择模型
- **策略演化**：观察优化
- **全景广播**：冷酷电子音播报
