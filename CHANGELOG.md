# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-20

### Added
- 支持 19 种创意选项（18 种创意风格 + 原时代风格）
- 支持多选 9 种杂志
- 一次性生成所有组合（风格 × 杂志 × 年代）
- 保持重新生成功能正常
- 保持历史记录保存/加载功能

### Fixed
- 修复 API Key 初始化逻辑，支持从 localStorage 读取
- 简化 Footer 布局为单行
- 修复 TypeScript 编译错误
- 增强人脸相似度功能

## [0.0.1] - 初始版本

- 基础杂志封面生成功能
- 多年代选择（1920s-2100s）
- 多杂志风格支持
- 历史记录功能
